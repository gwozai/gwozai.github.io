方法一：使用官方安装脚本（推荐）

1 安装步骤

步骤 1：下载并执行安装脚本

使用 curl 下载并执行安装脚本（推荐）：

curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash

或者使用 wget：

wget -qO- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash

步骤 2：配置环境变量

安装脚本会自动添加到 ~/.zshrc，以下是添加的内容：

export NVM_DIR="$HOME/.nvm"
[ -s "$NVM_DIR/nvm.sh" ] && \. "$NVM_DIR/nvm.sh"  # This loads nvm
[ -s "$NVM_DIR/bash_completion" ] && \. "$NVM_DIR/bash_completion"  # This loads nvm bash_completion

步骤 3：重新加载配置文件

source ~/.zshrc

步骤 4：验证安装

nvm --version

× 卸载步骤

步骤 1：删除 NVM 目录

rm -rf "$HOME/.nvm"

步骤 2：从 ~/.zshrc 删除配置

手动删除以下行：

export NVM_DIR="$HOME/.nvm"
[ -s "$NVM_DIR/nvm.sh" ] && \. "$NVM_DIR/nvm.sh"
[ -s "$NVM_DIR/bash_completion" ] && \. "$NVM_DIR/bash_completion"

步骤 3：重新加载配置文件

source ~/.zshrc

方法二：使用 Homebrew 安装

1 安装步骤

步骤 1：使用 Homebrew 安装 nvm

brew install nvm

步骤 2：创建 NVM 目录

mkdir ~/.nvm

步骤 3：配置环境变量

将以下配置添加到 ~/.zshrc：

# NVM configuration
export NVM_DIR="$HOME/.nvm"
[ -s "$(brew --prefix)/opt/nvm/nvm.sh" ] && \. "$(brew --prefix)/opt/nvm/nvm.sh"
[ -s "$(brew --prefix)/opt/nvm/etc/bash_completion.d/nvm" ] && \. "$(brew --prefix)/opt/nvm/etc/bash_completion.d/nvm"

步骤 4：重新加载配置文件

source ~/.zshrc

步骤 5：验证安装

nvm --version

× 卸载步骤

步骤 1：使用 Homebrew 卸载 nvm

brew uninstall nvm

步骤 2：删除 NVM 目录

rm -rf ~/.nvm

步骤 3：删除配置文件

从 ~/.zshrc 中删除 NVM 相关配置

source ~/.zshrc
🔧

常见问题排查

问题 1：命令未找到

• 确保已经重新加载了配置文件：source ~/.zshrc

• 检查 PATH 环境变量是否正确设置

问题 2：权限问题

• 确保 ~/.nvm 目录存在且有正确的权限

• 检查用户对相关目录是否有读写权限

问题 3：路径中包含空格

• 如果使用 Homebrew 安装时遇到路径空格问题，建议使用官方安装脚本方式

• 确保路径中的空格已正确转义

问题 4：版本问题

• 使用 nvm ls-remote 查看可用的 Node.js 版本

• 使用 nvm current 查看当前使用的版本
